* (coquand, paulin) inductively defined types

*** 1.1 Generalized inductive types

    Given an expression `F: (X: U) -> U`
    `strictly_positive_p(F)` is defined to be true
    if one of the following holds

    1. not_occur(X, F(X))
    2. F(X) = X
    3. F(X) = (k: K) -> (G(X))
       and not_occur_p(X, K)
       and strictly_positive_p(G)
